Stretch Questions

    1. What is the difference between an inline element and a block element?
	A block element displays normally while an inline element only takes up what space is necessary. For example
		if you have a div element as a block element and it's a paragraph, it would display normally. The 
		inline element would be a span element to highlight a word.
    2. What happens when an element is positioned absolutely? 
	It has a fixed position. It makes other elements act like it doesn't exist.
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
	You set the margin to auto.
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default.
	Block: <div>, <p>, and <section> 
	Inline: <span>, <img>, <a href="">
	Inline-block: <nav>
    5. In your own words, explain the box model. What is the fix for the box model? 
	The box model is like a box that surrounds elements. It consists of margins, padding, and borders. You can manipulate each 
	of these to be able to position your elements and add borders around them. The fix for the box model is box-sizing:border-box,
	with this you can easily adjust your padding and margins more easily.
